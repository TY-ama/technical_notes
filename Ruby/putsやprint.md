# String Operations
- [[puts]]
- **\n**: 改行
- **print "今の値段：#{value * 0.7}"**
	- 文字列の中に変数や式を埋め込み可能
- **#{format("%5d", fund)}**
	- 絶対に5文字分使う
- **puts "" + gets, puts "" , gets**
	- 二行で表示
- **text = "hello"**
- message = text
	- textとmessageの両方に"hello"が紐づけられる(ポインタに渡したときみたいな)
- **text.delete!("o")**
	- ！あり = text自体が変更　
	- ！なし = 新しい文字列返すだけ
